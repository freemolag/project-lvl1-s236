## Setup

```
$ npm i brain-games-dmitry-sorokin -g
```

## Run calc game

```
$ brain-calc
```

## Run even game

```
$ brain-even
```

## Run gcd game

```
$ brain-gcd
```

## Run balance game

```
$ brain-balance
```

## Run progression game

```
$ brain-progression
```

## Run prime game

```
$ brain-prime
```
